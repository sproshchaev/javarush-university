# Java Objects, Constructors, and Encapsulation Examples

Примеры кода для вебинара по основам Java: устройство Java-программы, соглашения о стиле (Code Conventions), объекты, конструкторы, инициализация переменных, геттеры и сеттеры, а также сравнение объектов.
Каждый файл соответствует одному слайду и содержит исполняемый пример.
