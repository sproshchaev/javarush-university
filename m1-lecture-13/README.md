Java Static Members and Nested Classes Examples
Примеры кода для вебинара по теме статических переменных, методов и вложенных классов в Java.
Рассматриваются различия между статическими и нестатическими членами, особенности статических вложенных классов (static nested class) и нестатических внутренних классов (inner class), а также порядок инициализации.

Каждый файл соответствует одному слайду и содержит исполняемый пример.

Слайды и соответствующие классы:
1. Статические переменны [Slide03_StaticVariableExample.java](src/main/java/com/javarush/example/Slide03_StaticVariableExample.java)
2. Разница между статическими и нестатическими переменными [Slide04_StaticVsInstanceVariableExample.java](src/main/java/com/javarush/example/Slide04_StaticVsInstanceVariableExample.java)
3. Статические методы [Slide05_StaticMethodExample.java](src/main/java/com/javarush/example/Slide05_StaticMethodExample.java)
4. Сравнение статических и обычных методов [Slide06_StaticVsInstanceMethodComparison.java](src/main/java/com/javarush/example/Slide06_StaticVsInstanceMethodComparison.java)
5. Статические и нестатические внутренние классы [Slide07_NestedClassExample.java](src/main/java/com/javarush/example/Slide07_NestedClassExample.java)
6. Использование статического вложенного класса [Slide08_StaticNestedClassExample.java](src/main/java/com/javarush/example/Slide08_StaticNestedClassExample.java)
7. Особенности статических вложенных классов [Slide09_StaticNestedClassFeaturesExample.java](src/main/java/com/javarush/example/Slide09_StaticNestedClassFeaturesExample.java)
8. Порядок инициализации внешнего и статического вложенного классов [Slide10_InitializationOrderExample.java](src/main/java/com/javarush/example/Slide10_InitializationOrderExample.java)
