# Потоки ввода-вывода в Java Examples

Примеры кода для вебинара по потокам ввода-вывода в Java.
Каждый файл соответствует группе слайдов и содержит исполняемый пример.

## Слайды и соответствующие классы:

1. Слайды 3-4: Базовые потоки и различия между байтовыми и символьными потоками [Slide03_04_ByteVsCharStreams.java](src/main/java/com/javarush/example/Slide03_04_ByteVsCharStreams.java)  
2. Слайды 7-9: Цепочки потоков, InputStreamReader и BufferedReader [Slide07_09_StreamChaining.java](src/main/java/com/javarush/example/Slide07_09_StreamChaining.java)  
3. Слайды 11-13: Потоки вывода, Writer и BufferedWriter [Slide11_13_OutputAndBuffering.java](src/main/java/com/javarush/example/Slide11_13_OutputAndBuffering.java)  
4. Слайды 14-19: Потоки в памяти - ByteArrayInputStream и ByteArrayOutputStream [Slide14_19_InMemoryStreams.java](src/main/java/com/javarush/example/Slide14_19_InMemoryStreams.java)
