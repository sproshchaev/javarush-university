# Exceptions and Resources Management in Java Examples

Примеры кода для вебинара по исключениям и управлению ресурсами в Java.  
Каждый файл соответствует одному слайду и содержит исполняемый пример.

## Слайды и соответствующие классы:

1. Создание своих исключений [Slide03_CustomExceptionTransformation.java](src/main/java/com/javarush/example/Slide03_CustomExceptionTransformation.java)  
2. Stack trace: получение цепочки вызовов методов [Slide04_StackTraceAnalysis.java](src/main/java/com/javarush/example/Slide04_StackTraceAnalysis.java)  
3. StackTraceElement: детальная информация об элементах стека [Slide05_StackTraceElementDetails.java](src/main/java/com/javarush/example/Slide05_StackTraceElementDetails.java)  
4. Стек: структура данных LIFO и класс Stack [Slide06_StackOperations.java](src/main/java/com/javarush/example/Slide06_StackOperations.java)  
5. Вывод стек-трейса при обработке ошибок [Slide07_ExceptionStackTrace.java](src/main/java/com/javarush/example/Slide07_ExceptionStackTrace.java)  
6. Внешние ресурсы и метод close() [Slide08_ExternalResourcesClose.java](src/main/java/com/javarush/example/Slide08_ExternalResourcesClose.java)  
7. Try-with-resources: автоматическое управление ресурсами [Slide09_TryWithResources.java](src/main/java/com/javarush/example/Slide09_TryWithResources.java)  
8. Несколько переменных одновременно в try-with-resources [Slide10_MultipleResources.java](src/main/java/com/javarush/example/Slide10_MultipleResources.java)  
9. Интерфейс AutoCloseable: требование для автоматического закрытия [Slide11_AutoCloseableInterface.java](src/main/java/com/javarush/example/Slide11_AutoCloseableInterface.java)
