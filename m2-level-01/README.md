# OOP: Encapsulation and Polymorphism Examples

Примеры кода для вебинара по принципам ООП: инкапсуляции и полиморфизму в Java.  
Каждый файл соответствует теме слайда и содержит исполняемый пример.

## Слайды и соответствующие классы:

### Основные принципы ООП
1. **ООП — основные принципы** (Слайд 3)  
   *Только теория — четыре "кита" ООП: абстракция, инкапсуляция, наследование, полиморфизм.*

2. **Абстракция** (Слайд 4)  
   *Только теория — искусство выделять главное и отбрасывать второстепенное.*

### Инкапсуляция
3. **Инкапсуляция: принципы и преимущества** (Слайды 5, 9-11)  
   [Slide05_EncapsulationDemo.java](src/main/java/com/javarush/example/Slide05_EncapsulationDemo.java)  
   *Полная демонстрация инкапсуляции: сокрытие данных, геттеры/сеттеры, валидация, сокрытие реализации.*

### Наследование и полиморфизм
4. **Наследование** (Слайд 6)  
   *Интегрировано в пример полиморфизма.*

5. **Полиморфизм** (Слайды 7, 12-13)  
   [Slide06_PolymorphismDemo.java](src/main/java/com/javarush/example/Slide06_PolymorphismDemo.java)  
   *Комплексный пример наследования и полиморфизма: переопределение методов, приведение типов, динамическая диспетчеризация, использование super.*
