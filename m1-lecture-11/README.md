# Java Basics and OOP Examples

Примеры кода для вебинара по основам Java: примитивные типы данных, приведение типов, объектно-ориентированное программирование (ООП) и другие фундаментальные концепции.
Каждый файл соответствует одному слайду и содержит исполняемый пример.
