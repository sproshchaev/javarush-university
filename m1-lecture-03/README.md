
# Java Conditional Operators Examples

Примеры кода для вебинара по условным операторам в Java.  
Каждый файл соответствует одному слайду и содержит исполняемый пример.

## Слайды и соответствующие классы:

1. Основы оператора `if-else` [Slide01_IfElse.java](src/main/java/com/javarush/example/Slide01_IfElse.java)
2. Пример работы `if-else` [Slide02_IfElseExample.java](src/main/java/com/javarush/example/Slide02_IfElseExample.java)
3. Пример использования блока команд [Slide03_BlockOfCommands.java](src/main/java/com/javarush/example/Slide03_BlockOfCommands.java)
4. Примеры сочетания `if-else` и блока команд [Slide04_IfElseBlocksExample.java](src/main/java/com/javarush/example/Slide04_IfElseBlocksExample.java)
5. Сокращённая форма `if` без `else` [Slide05_IfWithoutElse.java](src/main/java/com/javarush/example/Slide05_IfWithoutElse.java)
6. Последовательность `if-else if-else` [Slide06_IfElseIf.java](src/main/java/com/javarush/example/Slide06_IfElseIf.java)
7. Для чего нужен тип `boolean` [Slide07_BooleanType.java](src/main/java/com/javarush/example/Slide07_BooleanType.java)
8. Использование булевых переменных [Slide08_BooleanVariables.java](src/main/java/com/javarush/example/Slide08_BooleanVariables.java)
9. Операторы сравнения (`<`, `>`, `==`, `!=`, и т.д.) [Slide09_ComparisonOperators.java](src/main/java/com/javarush/example/Slide09_ComparisonOperators.java)

11. Различие между выражениями и командами [Slide11_ExpressionsAndStatements.java](src/main/java/com/javarush/example/Slide11_ExpressionsAndStatements.java)
12. Тернарный оператор [Slide12_TernaryOperator.java](src/main/java/com/javarush/example/Slide12_TernaryOperator.java)
