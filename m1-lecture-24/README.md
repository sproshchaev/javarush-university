# Java NIO Examples

Примеры кода для вебинара "Потоки ввода-вывода. Часть 2".
Каждый файл соответствует группе слайдов и содержит исполняемый пример.

## Слайды и соответствующие классы:
1. Создание и методы класса Path, абсолютные и относительные пути, класс Paths [Slide06_PathExample.java](src/main/java/com/javarush/example/Slide06_PathExample.java)  
2. Создание, чтение, запись, копирование файлов и работа с директориями [Slide10_FilesExample.java](src/main/java/com/javarush/example/Slide10_FilesExample.java)
