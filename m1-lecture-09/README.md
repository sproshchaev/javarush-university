# Strings in Java Examples

Примеры кода для вебинара по методам в Java.
Каждый файл соответствует одному слайду и содержит исполняемый пример.

## Слайды и соответствующие классы:
1. Константы [Slide03_Constants.java](src/main/java/com/javarush/example/Slide03_Constants.java)  
2. Область видимости переменных [Slide04_ScopeOfVariables.java](src/main/java/com/javarush/example/Slide04_ScopeOfVariables.java)  
3. Затенение переменных [Slide05_VariableShadowing.java](src/main/java/com/javarush/example/Slide05_VariableShadowing.java)  
4. Экранирование символов [Slide06_EscapeSequences.java](src/main/java/com/javarush/example/Slide06_EscapeSequences.java)  
5. String Pool [Slide07_StringPool.java](src/main/java/com/javarush/example/Slide07_StringPool.java)  
6. Методы класса [Slide08_StringMethods.java](src/main/java/com/javarush/example/Slide08_StringMethods.java)  
7. Сравнение строк [Slide09_StringComparison.java](src/main/java/com/javarush/example/Slide09_StringComparison.java)  
8. Поиск подстрок [Slide10_SubstringSearch.java](src/main/java/com/javarush/example/Slide10_SubstringSearch.java)  
9. Создание подстрок [Slide11_CreateSubstrings.java](src/main/java/com/javarush/example/Slide11_CreateSubstrings.java)
