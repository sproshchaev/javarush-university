
# Loops in Java Examples

Примеры кода для вебинара по циклам в Java.
Каждый файл соответствует одному слайду и содержит исполняемый пример.

## Слайды и соответствующие классы:

1. Пример работы цикла while [Slide03_WhileLoopExample.java](src/main/java/com/javarush/example/Slide03_WhileLoopExample.java)  
2. Цикл for: компактная запись повторяющихся действий [Slide04_ForLoopExample.java](src/main/java/com/javarush/example/Slide04_ForLoopExample.java)  
3. Цикл do-while: выполнение тела хотя бы один раз [Slide05_DoWhileLoopExample.java](src/main/java/com/javarush/example/Slide05_DoWhileLoopExample.java)  
4. Команда break: досрочный выход из цикла [Slide06_BreakExample.java](src/main/java/com/javarush/example/Slide06_BreakExample.java)  
5. Команда continue: пропуск текущей итерации (и типичная ошибка) [Slide07_ContinueExample.java](src/main/java/com/javarush/example/Slide07_ContinueExample.java)  
6. Вложенные циклы: цикл внутри цикла Slide08_NestedLoopsExample.java [Slide08_NestedLoopsExample.java](src/main/java/com/javarush/example/Slide08_NestedLoopsExample.java)
