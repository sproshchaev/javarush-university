# OOP: Abstract Classes, Overloading and Overriding Examples

Примеры кода для вебинара по продвинутым концепциям ООП в Java: абстрактным классам, перегрузке и переопределению методов.  
Каждый файл соответствует теме слайда и содержит исполняемый пример.
