# Arrays in Java Examples

Примеры кода для вебинара по массивам в Java.
Каждый файл соответствует одному слайду и содержит исполняемый пример.

## Слайды и соответствующие классы:

1. Создание массива [Slide04_CreateArrayExample.java](src/main/java/com/javarush/example/Slide04_CreateArrayExample.java)
2. Работа с ячейками массива [Slide05_ArrayCellsExample.java](src/main/java/com/javarush/example/Slide05_ArrayCellsExample.java)
3. Длина массива [Slide06_ArrayLengthExample.java](src/main/java/com/javarush/example/Slide06_ArrayLengthExample.java)
4. Массив типа [Slide07_StringArrayExample.java](src/main/java/com/javarush/example/Slide07_StringArrayExample.java)
5. Быстрая инициализация массива в Java [Slide08_QuickInitArrayExample.java](src/main/java/com/javarush/example/Slide08_QuickInitArrayExample.java)
