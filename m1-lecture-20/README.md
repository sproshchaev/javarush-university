# Exceptions in Java Examples

Каждый файл соответствует одному слайду и содержит исполняемый пример.

4. Перехват исключений: try-catch [Slide04_TryCatchBasics.java](src/main/java/com/javarush/example/Slide04_TryCatchBasics.java)  
5. Несколько блоков catch [Slide05_MultipleCatchBlocks.java](src/main/java/com/javarush/example/Slide05_MultipleCatchBlocks.java)  
6. Порядок блоков catch / Иерархия исключений [Slide06_CatchOrderHierarchy.java](src/main/java/com/javarush/example/Slide06_CatchOrderHierarchy.java)  
7. Выбрасывание исключений [Slide07_ThrowingExceptions.java](src/main/java/com/javarush/example/Slide07_ThrowingExceptions.java)
