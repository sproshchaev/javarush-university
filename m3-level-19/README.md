# Java Concurrent Examples

Примеры кода для вебинара "Многопоточность в Java. Concurrent".
Каждый файл соответствует группе слайдов и содержит исполняемый пример.

## Слайды и соответствующие классы:

1. AtomicInteger, AtomicBoolean, AtomicReference и CAS операции [Slide07_AtomicExamples.java](src/main/java/com/javarush/example/Slide07_AtomicExamples.java)
2. CopyOnWriteArrayList, ConcurrentHashMap, ConcurrentSkipListMap [Slide09_ConcurrentCollectionsExample.java](src/main/java/com/javarush/example/Slide09_ConcurrentCollectionsExample.java)
3. ConcurrentLinkedQueue, BlockingQueue, ArrayBlockingQueue, PriorityBlockingQueue [Slide13_ConcurrentQueuesExample.java](src/main/java/com/javarush/example/Slide13_ConcurrentQueuesExample.java)
4. CountDownLatch, Semaphore, CyclicBarrier, Exchanger, Phaser [Slide17_SynchronizersExample.java](src/main/java/com/javarush/example/Slide17_SynchronizersExample.java)
5. Future, Callable, ExecutorService, ThreadPoolExecutor, ScheduledExecutorService [Slide18_ExecutorsExample.java](src/main/java/com/javarush/example/Slide18_ExecutorsExample.java)
6. ReentrantLock, ReadWriteLock, Condition, StampedLock [Slide24_LocksExample.java](src/main/java/com/javarush/example/Slide24_LocksExample.java)
